Watermarker

A programme which takes a background image and then overlays a watermark image on top of it, utilising the module OpenCV.
There are custom options like the position of the watermark, its transparency, and its size, as well as allowing the user to specify any directory they choose for either image.

Noughts and Crosses

A Python game of Noughts and Crosses which takes place in the console. I use a numpy array to keep track of the board.
It has a singleplayer mode with an AI, adding a layer of complexity above a standard two-human-player game. The AI is relatively simple - if it has two in a row, it will complete it to win, otherwise it will block the player from a three in a row, otherwise it will pick a random cell - but it presents an unexpected challenge.
